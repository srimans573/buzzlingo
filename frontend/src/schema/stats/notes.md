# SCHEMA: STATS COLLECTED PER USER

loginStreak: The amount of times that the user has answered at least one question per day in a row. Resets whenever failed in a day between.

currentAnswerStreak: The amount of correct answers the user has had in a row. 

maxAnswerStreak: All time high answer streak.

timeSpent: The amount of time that the user has spent on the app. 

xp: The amount of experience points that the user has. 

videosWatched: The number of videos watched. Self-explanatory.

newWordsLearned: Debating whether or not to have this. (It'll be hard to track the new words specifically.)
